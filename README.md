# Linux Server Administration Labs

Practical Linux administration labs covering users, permissions, services, networking, security, storage, monitoring, troubleshooting, and automation across Ubuntu, RHEL, CentOS, and AlmaLinux.

## About This Repository

This repository documents hands-on Linux administration work in a structured, recruiter-friendly format. Each lab includes:

- Objective and scenario
- Supported distributions
- Commands with explanations
- Verification steps
- Security considerations
- Troubleshooting guidance
- Cleanup instructions
- Screenshot and evidence checklist

> **Important:** Run these labs only in a test VM, cloud lab instance, or non-production server.
